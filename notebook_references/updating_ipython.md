## Updating IPython


### Windows
```
C:\Anaconda>c:\Anaconda\Scripts\pip.exe install --upgrade http://umich-cscs530.s3.amazonaws.com/winter2015/ipython-cscs530-20150123.zip
...

C:\Anaconda>c:\Anaconda\Scripts\pip.exe install jsonschema
...

```

### Mac

```
$ pip install --upgrade --upgrade http://umich-cscs530.s3.amazonaws.com/winter2015/ipython-cscs530-20150123.zip
$ pip install jsonschema
```
